# Student Registration Form

A professional, responsive web-based registration form designed for student data collection. This project showcases my foundational skills in **HTML5** and **CSS3** layout.

## 🚀 Features
* **User-Friendly Interface**: Clean design with clear labels for easy data entry.
* **Responsive Layout**: Designed to look great on both desktop and mobile devices.
* **Semantic HTML**: Built using modern coding standards for better accessibility.

## 🛠️ Technologies Used
* **HTML5**: For page structure and content.
* **CSS3**: For custom styling, typography, and layout.
